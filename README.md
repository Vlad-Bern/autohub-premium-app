Документация: Автомобильный калькулятор AutoHub Premium
Интерфейс и логика для интеграции калькулятора автоимпорта в Telegram Mini App. Приложение рассчитывает финальную стоимость машины под ключ из Китая, Кореи и Японии с учетом таможенных пошлин, коммерческого утильсбора и внутренних расходов компании.

Функциональные особенности
Адаптация под тему Telegram. Код использует системные переменные мессенджера (--tg-theme-*). Приложение автоматически подстраивается под темную или светлую тему на любых клиентах, включая AyuGram, без принудительной инверсии цветов браузером.

Кастомные жесты (Свайп). Переключение стран и возраста машины реализовано на Pointer Events с фиксацией координат через матрицу трансформаций DOMMatrix. Это исключает зависание плашки и блокирует случайный скролл самой страницы на смартфонах.

Динамические диапазоны. При выборе страны ползунок цены автоматически меняет минимальные/максимальные лимиты, шаг и валюту под выбранный регион (Юани, Воны, Йены).

Актуальная математика пошлин. Расчет учитывает заградительные тарифы ФТС РФ для автомобилей старше 5 лет и повышенные коэффициенты утильсбора для двигателей большого объема.

Настройка параметров бизнеса (script.js)
Для изменения тарифов компании или актуализации курсов валют открывать HTML-код не нужно. Все переменные вынесены в начало файла script.js.

Настройка логистики и маржи
JavaScript
const CONFIG = {
    logistics: 250000, // Стоимость доставки авто в рублях
    margin: 75000,     // Комиссия компании в рублях
    defaults: {
        price: 165000, // Начальная цена при сбросе
        volume: 2000,  // Начальный объем двигателя
    },
};
Настройка валют и лимитов ползунка
Каждая страна описана отдельным объектом в массиве CURRENCY_DATA:

JavaScript
const CURRENCY_DATA = [
    { 
        name: "china", 
        rate: 13.42,       // Курс валюты к рублю
        symbol: "¥", 
        label: "Цена в Китае (¥)", 
        min: 50000,        // Минимальное деление ползунка
        max: 1000000,      // Максимальное деление ползунка
        step: 5000,        // Шаг ползунка
        defaultPrice: 165000 
    },
];
Интеграция с бэкендом (Для разработчиков)
Интерфейс полностью подготовлен к сбору данных. При нажатии на кнопку «Заказать этот автомобиль» (#submitOrderBtn) скрипт собирает текущее состояние калькулятора.

Разработчик бэкенда может настроить отправку этой информации в CRM-систему или через Telegram-бота в рабочий чат менеджеров.

Структура собираемых данных
При отправке заявки калькулятор формирует объект со следующими параметрами:

JSON
{
  "selected_country": "china",
  "car_price_local": 165000,
  "currency_symbol": "¥",
  "engine_volume_cc": 2000,
  "car_age_group": "До 3 лет (Новая)",
  "exchange_rate_rub": 13.42,
  "total_price_rub": 3908164
}

Развертывание и запуск
Убедитесь, что исходный файл style.scss скомпилирован в итоговый style.css.

Загрузите файлы (index.html, style.css, script.js) на любой хостинг с поддержкой HTTPS (Vercel, GitHub Pages или Netlify).

Перейдите в Telegram к боту @BotFather, создайте новое приложение командой /newapp, выберите своего бота и укажите URL-адрес вашего хостинга.
